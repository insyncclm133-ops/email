

## Add Help Widget Script

A single change to `index.html` to include the external help widget script before the closing `</body>` tag.

### Technical Details

**File: `index.html`**
- Add `<script src="https://go-in-sync.lovable.app/help-widget.js" data-source="Audi"></script>` just before the closing `</body>` tag, after the existing root div and main script.
